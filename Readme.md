# LinkVault

## Initial Repository Setup

### Initialize Git Repository
```bash
git init
git add .
git commit -m "first commit"
```

### Set Main Branch
```bash
git branch -M main
```
Renames the current branch to "main" (`-M` = `--move --force`)

### Connect to Remote and Push
```bash
git remote add origin https://github.com/prakhar060703/LinkVault.git
git push -u origin main
```
